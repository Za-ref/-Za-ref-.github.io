1: Describe the general process for executing addition, regardless of base.

Put both values one on top of another, and sarting from the lowest values of integers, add the values in a columm together while carrying any extra values to the next column over.

2: What special factors need to be considered when executing addition with octal numbers?

Octal numbers only go in groups of 8, so all sets can only add up to ad carry a value of 8 max with integers ranging from 0-7. Octal numbers go in sets of 3. Ex, 163 257.

3: What special factors need to be considered when executing addition with hexadecimal numbers?

Not only are there a total of 16 integers in hexadecimal systems, it goes from 0-9 to A-F. Be careful when adding to acknowledge the letters numerical values. Hexadecimal values go in groups of 2. Ex. F2, 3C

4: What special factors need to be considered when executing addition with binary numbers?

Binary numbers only have 0 and 1, so they must be in 2 integer groups. Binary numbers are also written in groups of 4. Ex. 0010 1101
